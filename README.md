# react-native-iframe-youtube-player

A simple youtube player.

## Installation

```sh
npm install react-native-iframe-youtube-player
```

## Usage

```js
import { multiply } from 'react-native-iframe-youtube-player';

// ...

const result = await multiply(3, 7);
```
